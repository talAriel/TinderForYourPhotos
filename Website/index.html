About the project

Tinder For Your Photos app was developed by Adi Schwartz & Tal Ariel under the supervision of Roman Sandler (Shutterfly).


Project Discription

Tinder app is famous for its "swipe" feature, allowing fast filtering of photo series. In this project, the students implemented a Tinder-like interface for fast like/dislike selection of single images. The project was extended for learning the user's taste in photos, which is important for Shutterfly to simplify and enrich the photo product creation process. The desired result is learning the "liked" photos parameters, and generates collages adjusted to the learned user preferences.

The Goal

- Develope an end-to-end mobile app and implementing a Tinder-like interface for fast like/dislike selection of single images or image collages.
- Learn the user's taste in photos.

Achievments

Android App

We implemented a Tinder-like app, designed as a Shutterfly app (as you can see in the Gallery section).

HTTP Server

We used Apache server to handle all the requests made by the app. The server handles the requests, operate on the data (which is stored in a MySQL database) and send a matching response.

Machine Learning

- In order to learn the user's taste we implemented an unsupervized Naive Bayes learning model.
- The user's actions labeled the photos.
- By using the knowledge we got from the user, we were able to choose all the best photos.

Tools & Enviroment

- The app was developed in Android Studio and implemented in Java.
- The server side was developed fully by us and written in PHP and Python.
- The data was stored in a MySQL database
- An external library for the tinder-like interface: PlaceHolderView by Janisher Ali
